Diseña un sistema web administrativo para un taller automotriz moderno, minimalista y profesional.
El sistema debe tener interfaz clara, organizada y enfocada en usabilidad.
Estilo limpio tipo dashboard SaaS, colores sobrios (azul oscuro, gris claro y blanco), botones con bordes redondeados, iconografía simple y diseño responsive.

🔹 Tipografía:

Helvetica
Jerarquía clara (H1, H2, body, labels)
